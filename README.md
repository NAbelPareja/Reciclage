# EcoHuanta 🌱

Aplicación web desarrollada con Angular para:

- Reportar problemas ambientales
- Mostrar centros de reciclaje
- Obtener ubicación automáticamente
- Enviar reportes por WhatsApp

## Tecnologías

- Angular
- TypeScript
- HTML
- CSS
- OpenStreetMap API

## Funciones

- Geolocalización
- Reportes ambientales
- Consejos ecológicos
- Mapas interactivos
